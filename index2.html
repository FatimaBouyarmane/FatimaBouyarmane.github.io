Phishing websites Detector using Machine Learning

The Phishing Website Detector is a machine learning-based tool designed to identify phishing websites using a Random Forest classifier trained on a dataset of phishing and legitimate URLs. Built with simplicity and effectiveness in mind, this project is ideal for users looking to enhance their online security.
